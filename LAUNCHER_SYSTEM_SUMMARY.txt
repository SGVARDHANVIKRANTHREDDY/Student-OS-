## 🛠️ TECHNICAL DETAILS

### Architecture
```
RUNSTUDENTOS.* (Entry Point)
    ├─→ Start Backend (npm run dev)
    ├─→ Wait for health check
    ├─→ Start Frontend (npm run dev)
    ├─→ Wait for health check
    ├─→ Create admin user (HTTP POST)
    ├─→ Verify services (HTTP GET)
    ├─→ Open browser
    └─→ Monitor & auto-restart loop
```

### Technology Stack
- **Python:** 3.8+
- **Node.js:** 18+
- **npm:** 8+
- **Subprocess:** Process management
- **urllib:** HTTP requests (no external deps needed)
- **webbrowser:** Browser automation

### Key Features
- No external Python dependencies
- Built-in error handling
- Process supervision
- Automatic cleanup
- Signal handlers for graceful shutdown

## 🐛 TROUBLESHOOTING

### Python not found?
```bash
# Install Python 3.8+ from python.org
# Add to PATH during installation
python --version  # Should show version
```

### Port already in use?
```bash
# Windows
taskkill /F /IM node.exe

# Mac/Linux
pkill node
```

### Backend won't start?
```bash
cd backend
npm install
PG_DISABLED=1 npm run dev
```

### Frontend won't start?
```bash
cd frontend
npm install
npm run dev
```
